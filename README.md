# Kubernetes 学习笔记

### 内容介绍

Kubernetes是谷歌开源的容器集群管理系统,是Google多年大规模容器管理技术Borg的开源版本,也是CNCF最重要的项目之一。

### 访问方式

这是个人学习Kubernetes的笔记，请点击下面的链接阅读:

- [在线阅读](https://skyao.net/learning-kubernetes/)：hugo格式 + docsy主题，界面清爽。托管于国内节点，有备案，国内访问速度很快
